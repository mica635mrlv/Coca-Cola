    O que é um repositório no Git e qual a sua importância?
        É uma ferramenta que ajuda a controlar e acompanhar as mudanças feitas em um arquivo ao longo do tempo, assim 
        evitando a perda de projetos e permitindo que as pessoas possam ver suas versões anteriores, e permite que várias 
        pessoas trabalhem no mesmo projeto.

    Qual a função do Git Bash?
        É uma ferramenta que permite usar comandos do Git no computador, ele ajuda a controlar versões de arquivos, criar 
        cópias, salvar mudanças e enviar atualizações para a nuvem, como no GitHub.

    O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
        Ele prepara os arquivos para serem salvos, no caso, commitados, a diferença entre os dois é que o git add <arquivo> 
        é usado para um único arquivo específico, já o git add . adiciona todos os arquivos alterados de uma vez.

    Por que utilizamos o ponto (.) no comando git add .?
        O ponto (.) significa que o usuário quer adicionar todos os arquivos alterados, assim adicionando todas as mudanças 
        de uma vez, sem precisar especificar cada arquivo.

    Explique o comando git commit e por que é importante criar uma mensagem descritiva.
        O comando git commit salva as mudanças no histórico do projeto, como se tirasse uma "foto" do estado atual dos 
        arquivos, sendo importante criar uma mensagem descritiva para lembrar o que foi alterado e entender as mudanças 
        feitas.

    Qual a função do comando git push?
        O comando git push envia as mudanças do seu computador para um repositório online, como por exemplo o GitHub, 
        para garantir que o projeto esteja salvo na nuvem e acessível para outros pessoas.

    Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
        É importante criar commits com frequencia pois ajuda a manter um histórico das mudanças adicionadas organizado, 
        facilita a correção de erros e permite voltar a versões anteriores se algo der errado.

    Explique a função das tags HTML e como elas estruturam uma página.
        As tags HTML são como "blocos" que tem a função de organizar uma página da web, sem as tags HTML a página seria 
        apenas um monte de texto sem nenhuma formatação.

    O que são atributos em uma tag HTML e qual a sua função?
        Os atributos são configurações extras que ao serem adicionadas as tags HTML tem a função de modificar seu comportamento.

    O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
        É uma ferramenta que ajuda a escrever código no HTML mais rápido, usando ele enquando se digita algum atalho o 
        editor automaticamente vai mostrar opções para completar o código.

    O que é o box-sizing no CSS e por que usamos ele no reset inicial?
        O box-sizing define como a largura e altura de um elemento são calculadas, no reset inicial, ele é utilizado como 
        "box-sizing: border-box;" fazendo com que a largura e altura dos elementos incluam as bordas e o padding.

    Por que é importante utilizar tags semânticas no HTML?
        As tags semânticas no HTML são importantes pois ajudam a organizar melhor o código e facilitam a leitura para 
        outros desenvolvedores.

    O que são media queries no CSS e qual a sua função?
        As media queries no CSS é uma ferramenta que permite aos desenvolvedores criar layouts responsivos, adaptando a 
        aparência da página para diferentes tamanhos de tela, como celulares, tablets, computadores, etc.

    Qual a função do GitHub Pages?
        O GitHub Pages tem a função de transformar um repositório do GitHub em um site, ele é muito usado para projetos, 
        portfólios e documentações.

    O que é o Flexbox no CSS e para que ele é utilizado?
        O Flexbox é um sistema do CSS que facilita o posicionamento e o alinhamento dos elementos dentro de um container, 
        sendo útil para criar layouts flexíveis e responsivos.

    Qual a diferença entre justify-content e align-items em um container Flexbox?
        O justify-content serve para alinhar os itens horizontalmente, enquanto o align-items alinha os itens verticalmente.

    Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
        O flex-grow define o quanto um elemento pode crescer dentro do container, ocupando o espaço disponível.

    Explique a diferença entre flex-direction: column; e flex-direction: row;.
        O row tem a função de organizar os elementos na horizontal, um ao lado do outro, enquanto o column organiza os 
        elementos na vertical, um abaixo do outro.
